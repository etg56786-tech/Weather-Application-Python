# Weather-Application-Python
This weather application is a Python GUI program that was made using PyQt5 along with OpenWeatherMap API.

**Weather Application Description**

This weather application has a custom-made interface with themed buttons and labels, as well as themed weather icons using emojis which changes depending on the weather condition.
There is also error handling when there are any invalid city names, or API related errors, and/or when the application cannot connect to the internet.

In this project, I have gained experience from many different programming aspects including but not limited to; APIs, GUI, exception handling, and object-oriented programming using Python.

FEATURES:

- Search for the weather based on city/place name
- Current temperature reading in Fahrenheit
- The ‘Feels Like’ temperature reading system
- Weather icons based on the live weather condition
- Country detection
- PyQt5 custom GUI with hover effects
- Specific error handling for invalid city names, API, and connection errors
